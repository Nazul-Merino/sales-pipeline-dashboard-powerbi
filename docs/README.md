# Documentation

This folder contains supporting documentation for the dashboard.

## Files

### dashboard_storyline.pdf

This document provides a detailed explanation of the dashboard design, business objectives, data preparation process, DAX modeling decisions, and the analytical storyline implemented across the two dashboard pages.

The document describes:

* Business requirements addressed by the dashboard.
* Power Query data preparation and transformation steps.
* Currency conversion methodology using exchange rates.
* KPI definitions and visualization rationale.
* Sales Pipeline Overview page.
* Pipeline Forecast page.
* Forecasting, volatility, and pricing-risk analyses.
* Executive storytelling approach used to guide decision-making.

The purpose of this document is to provide additional context on the design choices and analytical framework behind the dashboard.
