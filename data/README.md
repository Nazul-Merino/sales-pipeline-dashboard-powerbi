# Data

The original dataset used in this project is not included in this repository.

This dashboard was developed using data extracted from a sales CRM system, where each record represents a commercial opportunity moving through different stages of the sales pipeline. The dataset was used exclusively for dashboard development, data modeling, and business intelligence visualization purposes.

The main table contains the following fields:

* Opportunity ID
* Closed
* Won
* Stage
* Stage Duration
* Segments
* Region
* Opportunity Owner
* Opportunity Grade
* Probability (%)
* Age
* Amount Currency
* Amount
* Created Date
* Primary Reason
* Country
* Amount USD

In addition, a supporting lookup table named **Exchange Rate Table** was used to convert opportunity values from local currencies into USD. This table contains the following fields:

* Opportunity Currency
* Exchange Rate
